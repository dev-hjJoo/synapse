# Security Policy

Synapse는 현재 **문서 우선(docs-first)** 단계이며, 실행 코드는 아직 배포되지 않았습니다. 코드가 등록되면 이 정책은 확장됩니다.

## 지원 범위

| 구성 요소 | 상태 |
|----------|------|
| 문서·플래닝 | 활성 — 사실 오류·링크 문제는 일반 이슈로 |
| 코드 (`05_src/`) | 아직 없음 — 추후 고지 |
| 운영 서비스 | 아직 없음 — 추후 고지 |

## 취약점 신고

코드·서비스가 추가된 뒤, 보안 관련 이슈는 **공개 이슈로 등록하지 말고** 유지보수자에게 비공개로 알려 주세요.

현재는 비공개 신고 채널이 구성되어 있지 않습니다. 필요 시 저장소 프로필의 공개 연락처를 통해 암호화된 이메일로 조정하겠습니다. 연락 수단이 정비되는 대로 이 섹션에 업데이트하겠습니다.

## 개인정보·리서치 데이터

- 사용자 인터뷰 원본 데이터(녹취·노트·참여자 명단)는 저장소에 커밋되지 않습니다. [`.gitignore`](.gitignore) 에 해당 경로가 차단되어 있습니다.
- 실수로 PII가 커밋된 경우 즉시 이슈를 열어 주시면 히스토리에서 제거 절차를 진행하겠습니다.
