# Contributor Covenant Code of Conduct

## 우리의 약속 (Our Pledge)

열린 환경을 만들기 위해, 커뮤니티 구성원·기여자·유지보수자는 나이·체격·장애·민족·성별·성적 지향·신경다양성·인종·종교·경험 수준·교육 배경·사회경제적 지위와 관계없이 모든 참여자에게 **괴롭힘 없는 경험(harassment-free experience)** 을 제공할 것을 서약합니다.

## 우리의 기준 (Our Standards)

긍정적인 환경을 만드는 행동:

- 사람들을 존중하고 환영하는 언어 사용
- 서로 다른 관점과 경험을 존중
- 건설적인 비판을 정중하게 주고받기
- 커뮤니티에 가장 좋은 것에 집중
- 다른 커뮤니티 구성원에 대한 공감 표현

허용되지 않는 행동:

- 성적 언어·이미지, 원치 않는 성적 관심·접근
- 모욕·경멸 · 인신공격·정치적 공격
- 공공·사적 괴롭힘
- 명시적 허락 없이 타인의 개인정보(주소·이메일 등) 공개
- 전문 환경에서 부적절하다고 합리적으로 판단되는 기타 행동

## 유지보수자의 책임

프로젝트 유지보수자는 허용 가능한 행동의 기준을 명확히 하고, 부적절한 행동에 대해 공정하고 적절한 시정 조치를 취할 책임이 있습니다.

## 적용 범위

이 행동 강령은 프로젝트의 모든 공간(이슈·PR·토론·문서)과 프로젝트를 대표하여 공개된 자리에 모두 적용됩니다.

## 신고

학대·괴롭힘·기타 부적절한 행동은 **저장소 이슈에 직접 적지 말고**, 유지보수자에게 직접 연락해 주세요. 신고는 신중히 처리되며 신고자의 신원은 보호됩니다. 유지보수자 연락처는 저장소 프로필을 참고하세요.

## 출처

이 행동 강령은 [Contributor Covenant](https://www.contributor-covenant.org) v2.1 을 바탕으로 작성되었습니다.
