# 03_design — 디자인

Synapse의 **정보 구조(IA)**, **상호작용 플로우**, **비주얼 시스템**.

## 구성 (Phase 3에서 채워짐)

- `design_principles.md` — 디자인 원칙 (ex. "읽기에 방해되지 않게", "AI는 조력자이지 권위자가 아니다")
- `ia.md` — 정보 구조
- `user_flows.md` — 주요 플로우 (임포트·이해 대화·다음 스텝)
- `wireframes/` — 와이어프레임 (이미지 또는 Figma 링크 참조)
- `design_system/` — 토큰·컴포넌트 사양
- `prototypes.md` — Figma·실기기 프로토타입 링크

## 원칙 (초안)
1. **Reader-first** — 본문을 가리거나 스크롤을 뺏지 않는다.
2. **Progressive disclosure** — 필요할 때 필요한 깊이만.
3. **AI is a collaborator, not an authority** — 출처·신뢰도를 항상 가시화.
4. **Same grammar across surfaces** — 웹·확장·모바일에서 상호작용 문법 공유.
