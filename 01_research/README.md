# 01_research — 리서치

사용자·시장·경쟁에 대한 **생각의 원재료**. 여기서 나온 통찰은 `02_product/`의 PRD·비전에 반영됩니다.

## 구성 (Phase 1에서 채워짐)

- `personas.md` — 페르소나 2–3개
- `competitive_landscape.md` — 경쟁·대체 도구 분석
- `user_journey.md` — 현재 "as-is" 여정과 간극 지점
- `problem_statement.md` — 수렴된 문제 진술문
- `user_interviews/` — 인터뷰 원본 노트 (익명 ID `P01`, `P02` …)
