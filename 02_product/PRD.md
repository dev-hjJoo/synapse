## 4. 기능 스펙

### 4.1 논문 임포트·파싱
- arXiv DOI·URL·PDF 업로드
- 구조 추출: 제목·저자·섹션·수식·표·참고문헌
- 실패 시 폴백 경로

### 4.2 대화형 이해 에이전트
- 질문·답변 (출처 하이라이트 필수)
- 사용자 배경지식 레벨 조정 (beginner / practitioner / expert)
- 수식·다이어그램 설명

### 4.3 관련 자료 탐색
- 참고문헌·피인용 그래프 탐색
- 웹 검색 (블로그·영상·구현)
- 인용 가능한 근거와 출처 표시

### 4.4 인사이트 캡처
- 하이라이트·노트·태그
- 개인 지식 그래프화 (장기)

### 4.5 다음 스텝 (Action Bridge)
- **구현 보조**: 최소 재현 코드 스캐폴딩
- **아티팩트 생성**: 랩 미팅용 슬라이드, 블로그 드래프트, 팀 공유용 요약
- **후속 탐색**: 다음에 읽을 논문 제안, 질문 카드

### 4.6 공유·협업 _(후순위)_
- 노트·하이라이트 공개/비공개
- 팀 라이브러리

## 5. 비기능 요구사항
- **성능**: p50 응답 ≤ 2s, p95 ≤ 6s (스트리밍 첫 토큰 기준)
- **품질**: Faithfulness ≥ 0.85, Citation Accuracy ≥ 0.90 (v1 목표)
- **프라이버시**: 사용자 업로드 문서·노트는 암호화. 기본은 비공개.
- **접근성**: WCAG 2.1 AA 준수 목표
- **국제화**: 한국어 우선, 영어 병행 (논문이 영어 중심이므로)

## 6. 출시 범위

### MVP (Phase 4)
- 한 가지 end-to-end 플로우 완결: **PDF 업로드 → 대화형 이해 → 다음 스텝 1개 제안(구현 스캐폴딩 또는 후속 논문)**
- 인증 · 개인 라이브러리 · 기본 노트

### v1
- 웹 검색·인용 그래프 통합
- 아티팩트 생성 (슬라이드·요약)
- 다국어 UI

### v2+
- 팀 공유, 공개 링크
- 브라우저 확장 (웹 어디서나 논문 저장)
- 모바일 리더
